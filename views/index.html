About ARGAPI

ARGAPI (pronounced argh-app-ee) is an API of managing Alternate Reality Game codes and resources.

How to use

  1. Open an account by accessing /login

  2. Create a new campaign by making a POST request to /campaign. You will need to provide:

    1. a campaign title,
    2. a base_url (used by the public when accessing information and resources about your campaign),
    3. an (optional) short summary of the campaign,
    4. and open_at and close_at times for when the campaign goes public and concludes.
    Once you have done that, the response will contain the _id to access your campaign.

    You can edit your campaign with a PUT request to /campaign/{campaign id} with the same data used at creation.

  3. Next, create a new resource for users to discover by making a POST request to /campaign/{campaign id}/resource. The request body must contain a JSON object of your data.

  4. Once you have done that, create a code for users to access the resource with. Do this with a POST request to /campaign/{campaign id}/code with the following:

    1. code: 1-31 character code used in the URL along with the campaign base_url to access the resource.
    2. target_resource: The id of the resource to be accessed.
    3. valid_after: Optional. A time after which the code can be publically used. Useful for time-gated content.
    4. valid_until: Optional. The time after which the code will cease to work.

  5. Now that all that has been done, you can access your resource by navigating to /{campaign base url}/{code}!
